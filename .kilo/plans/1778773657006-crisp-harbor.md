# Create Poll & Create Outfit Post - Design Alignment Plan

## Status: ✅ Completed

## Overview
Updated `create-poll.component` and `create-outfit-post.component` (HTML, SCSS, TS) to match the design specs in `social desgin/creratepoll.html` and `social desgin/createoutfitpost.html`.

## Changes Made

### create-poll.component
- **Layout**: Restructured from single-column Material cards to two-column grid (options left, config right)
- **Header**: Simplified to back button + title only (removed Community Feed button)
- **Stepper**: Added 3-step wizard (Select Outfits → Configure Poll → Preview & Publish)
- **Options**: Added drag handles, image placeholders, edit/delete actions, counter display
- **Config**: Replaced expiry date with Poll Duration select, added Privacy Settings cards
- **Buttons**: Changed to Cancel (outline) + Preview & Publish (primary with arrow)
- **Colors**: Updated to `#ff6fa8` primary, `#fff9fb` background, Inter font
- **TS**: Removed unused imports, simplified image upload logic, added privacy/duration fields

### create-outfit-post.component
- **Layout**: Restructured as modal/overlay with dark backdrop
- **Header**: Updated to "Share to Feed" title matching design
- **Outfit Preview**: Added selected outfit preview with image + info
- **Form**: Updated with proper sections for outfit selection, caption, privacy, and tag friends
- **Privacy**: Changed to card-based toggle (Public/Followers Only)
- **Tags**: Added tag badges with remove functionality
- **Buttons**: Updated to Cancel (outline) + Share Post (primary with send icon)
- **Colors**: Updated to match design palette with `#ff6fa8` primary
- **TS**: Removed unused imports (MatSelectModule, MatFormFieldModule, MatInputModule, MatCardModule), added tag management
